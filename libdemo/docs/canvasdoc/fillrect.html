fillRect(x,y,width,height)

绘制矩形并填充,但不描边 

  ctx.fillRect(x,y,width,height)

参数

Number 矩形左上顶点x坐标

Number 矩形左上顶点y坐标

Number 矩形宽度

Number 矩形高度


测验

矩形(50,20) 230 105.

这是个简便的画填充矩形的方法
