fillStyle

填充样式,默认是#000.它的值可以是颜色值(color),渐变对象(gradient),纹理填充对象(pattern). 

  ctx.fillStyle = '#225588';

color

颜色值1 fillStyle = 'RGB(64,128,192)'

颜色值2 fillStyle = 'RGBA(64,128,192,.6)'

颜色值3 fillStyle = 'HSL(360,50%,50%)'

颜色值4 fillStyle = 'HSL(360,50%,50%,.6)'

颜色值5 HEX fillStyle = '#225588'
