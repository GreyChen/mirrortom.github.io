CanvasRenderingContext2D

canvas 2d的绘制对象,绘制的相关方法与属性在此对象上.以下代码获取此对象.此所谓绘画"上下文对象"

2d对象的绘画效果表现在其所在的dom上.

  var canvas = document.querySelector('canvas');
  var context = canvas.getContext('2d');
