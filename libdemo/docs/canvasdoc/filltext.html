fillText(text, x, y [,maxWidth])

绘制文本内容 

  ctx.fillText(text, x, y [,maxWidth])

参数

string 要绘画的文本内容

Number 文本左上顶点x坐标

Number 文本左上顶点y坐标

Number 文本最大宽度,如果设定了,当文本超过设定宽度时,会压缩文本
