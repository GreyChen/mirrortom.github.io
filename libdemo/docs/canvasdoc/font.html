font

字体样式.在使用fillText()方法绘画文字时,通过此属性设置文字大小和字体,类似于css的font属性 

  ctx.font = '20px Microsoft YaHei,-apple-system,BlinkMacSystemFont,sans-serif,宋体';

测验

字体1 font = '24px 宋体'

字体2 font = '20px sans-serif'

字体3 font = '20px Microsoft YaHei,-apple-system,BlinkMacSystemFont,sans-serif,宋体'
